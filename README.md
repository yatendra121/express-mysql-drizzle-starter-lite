# express-drizzle-starter-lite

A starter template for building REST APIs with Express, Drizzle ORM, and MySQL.

## API Documentation

### `GET /`

Server is running.

### `GET /all-users`

it will fetch all users records.

## Running the App

Install the dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```
